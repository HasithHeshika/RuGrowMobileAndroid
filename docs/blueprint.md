# **App Name**: RuGrow Mobile

## Core Features:

- Real-Time Temperature Display: Display real-time temperature data from the plant environment.
- Soil Moisture Display: Display real-time soil moisture levels.
- Last Watered Date: Record and display the date and time the plant was last watered.
- Last Harvested Date: Record and display the date and time the plant was last harvested.
- Manual Watering Control: Allow users to manually trigger the watering system for a specific plant using plant ID. Transmits appropriate watering signals to the plants.

## Style Guidelines:

- Primary color: Deep green (#347C0A), symbolizing plant life and growth.
- Background color: Light beige (#F5F5DC), providing a neutral and natural backdrop.
- Accent color: Earthy brown (#8B4513) for buttons and highlights related to soil and harvesting.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Use icons representing temperature, moisture, water droplets, and harvested peppers.
- Clean and intuitive layout, prioritizing real-time data display and easy access to manual controls.
- Subtle animations for data updates and user interactions.
